Good Links

  • Github
  • StackOverFlow
  • Dev

Taking on why I chose these sites I will start at the top with Github. Github is version controll and code sharing site. Long ago it was instilled in me how important it is to keep an offsite coppy of your code and you can do that for free with Github. Also I should mention that for the modern developer your public Github is a key part of showing your work.

Moving onto StackOverFlow, oh god I love this site so much! Many times I have been trying to figure out how to solve a problem and found an answer on StackOverFlow. It is a site where programers can post tech questions about an issue their facing and get free help for the comunity of other programers. I can almost guarantee that at some point using StackOverFlow will save you a ton of time.

Lastly Dev is a socail media site just for developers and programers, it has a real strong comunity of people that want to talk about coding and are very happy to help out us newbs.
